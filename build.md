# 植物大战僵尸 - 游戏概念设计文档 (v1.0)

## 1. 主界面设计

主界面以马年新春为主题，营造欢乐、卡通的氛围。

*   **背景**： 白天草坪，远处有喜庆的红灯笼、剪纸风格的马年吉祥物“小马奔腾”剪影，以及绽放的烟花。
*   **核心UI元素**：
    *   **「开始游戏」按钮**： 醒目按钮，点击进入关卡选择界面。
    *   **「关卡选择」**： 以一卷展开的卷轴形式呈现已解锁和未解锁的关卡图标。
    *   **「商店」**： 图标为一个戴着小马头套的疯狂戴夫商店招牌，用于购买植物、道具和升级。
    *   **「植物图鉴」**： 收集并查看所有已解锁植物的详细信息、趣事和获取方式。
    *   **「僵尸图鉴」**： 收集并查看所有遭遇过的僵尸的详细信息、弱点和趣事。
    *   **「设置」**： 音效、音乐、语言、游戏帮助等选项。
    *   **「退出」**： 退出游戏。

## 2. 游戏玩法

*   **核心循环**： 收集阳光 -> 在棋盘格草坪上种植植物 -> 植物自动攻击来袭的僵尸 -> 保护房屋（位于屏幕最左侧）不被僵尸进入。
*   **基本操作**： 鼠标点击/触摸屏操作。点击收集阳光，从屏幕下方的植物卡牌区选择植物，再点击草坪格子进行种植。
*   **胜利条件**： 成功抵御一波又一波的僵尸攻势，直至关卡倒计时结束或击败最终旗帜僵尸。
*   **失败条件**： 任一僵尸成功突破防线，进入房屋。
*   **资源管理**：
    *   **阳光**： 主要资源，通过种植向日葵/阳光菇生产，或从天上掉落获取。
    *   **植物卡牌冷却**： 每种植一株植物后，该植物卡牌会进入短时冷却，无法立即再次使用。
*   **关卡流程**： 每关由多个常规僵尸波次和一个标志性的“旗帜僵尸波次”（大量僵尸出现的提示）组成。击败最终旗帜僵尸或存活至倒计时结束即获胜。

## 3. 植物设计 (至少5种)

每种植物卡牌消耗不同阳光，并有独立的冷却时间。

| 植物名称 | 阳光消耗 | 冷却时间 | 功能描述 | 升级方向 (示例) |
| :--- | :--- | :--- | :--- | :--- |
| **向日葵** | 50 | 短 | 基础经济植物。每隔一段时间产生25阳光。 | **双生向日葵**： 生产阳光的间隔缩短，单次产量提升至50。 |
| **豌豆射手** | 100 | 中等 | 基础攻击植物。向前方直线发射豌豆，每次攻击造成1点伤害。 | **双发射手**： 一次射出两颗豌豆，攻速翻倍。 |
| **樱桃炸弹** | 150 | 长 | 一次性消耗品植物。种植后立即爆炸，对3x3范围内所有僵尸造成巨大伤害。 | **连环樱桃**： 爆炸后留下两个小樱桃，延迟后再次爆炸。 |
| **坚果墙** | 50 | 中等 | 防御植物。拥有高额生命值，阻挡僵尸前进。僵尸会停下来啃食它。 | **高坚果**： 生命值大幅提升，并能阻挡撑杆跳僵尸等特殊僵尸。 |
| **寒冰射手** | 175 | 中等 | 攻击植物。发射冰豌豆，造成1点伤害并附带减速效果（持续数秒）。 | **寒冰瓜**： 攻击变为投掷冰瓜，造成范围伤害和减速。 |

## 4. 僵尸设计 (至少5种)

*   **基础僵尸**：
    *   **普通僵尸**： 生命值低，速度慢，无特殊能力。最常见的敌人。
    *   **路障僵尸**： 头上戴有路障，生命值是普通僵尸的两倍。路障被击破后变为普通僵尸。
    *   **撑杆跳僵尸**： 速度较快，持有一个撑杆。遇到第一个植物时会撑杆跳过，但跳过后会丢失撑杆并减速。
    *   **铁桶僵尸**： 头上戴有铁桶，生命值极高。铁桶被击破后变为普通僵尸。
    *   **旗帜僵尸**： 举着一面小旗子，生命值与普通僵尸相同。其出现标志着一大波僵尸即将来袭，击败他通常代表关卡胜利在望。

## 5. 关卡设计 (3关示例)

### **关卡 1-1： 草坪初战 (白天)**
*   **目标**： 熟悉基本操作，成功抵御5波僵尸（含1个旗帜僵尸波次）。
*   **地形**： 标准的5x9草坪，无特殊地形。
*   **解锁植物**： 向日葵、豌豆射手。
*   **出现僵尸**： 普通僵尸、路障僵尸。
*   **设计要点**： 教学关卡，节奏缓慢，阳光充足，让玩家建立“种植向日葵 -> 收集阳光 -> 布置豌豆射手”的基础循环。

### **关卡 1-2： 迷雾之夜**
*   **目标**： 在夜间和迷雾环境下，抵御7波僵尸（含2个旗帜僵尸波次）。
*   **地形**： 5x9草坪，**夜晚场景**（初始阳光为0，需依靠新植物生产），**覆盖迷雾**（遮挡右侧部分视野，僵尸进入可见范围才显示）。
*   **解锁植物**： 阳光菇、小喷菇。
*   **出现僵尸**： 普通僵尸、路障僵尸、撑杆跳僵尸。
*   **设计要点**： 引入夜晚资源获取机制和视野限制，考验玩家对新植物的运用和迷雾中的布局预判。

### **关卡 1-3： 倾斜屋顶**
*   **目标**： 在倾斜屋顶地形上，抵御9波僵尸，击败最终巨型旗帜僵尸。
*   **地形**： 屋顶场景，棋盘格呈斜坡状。屋顶上有**花盆位置**，只有将植物种在预先放置或玩家放置的花盆中才能生效。
*   **解锁植物**： 卷心菜投手。
*   **出现僵尸**： 普通僵尸、路障僵尸、铁桶僵尸、气球僵尸（需用仙人掌等对空植物应对，此处作为隐藏挑战）。
*   **设计要点**： 引入复杂地形限制，要求玩家战略性地使用花盆，并应对新增的强力僵尸。最终波次僵尸数量多，强度高，是本章的小挑战。

## 6. 背景音乐 (BGM) 与音效

*   **主界面音乐**： 轻松、欢快、带有马年喜庆元素的卡通风格Loop音乐。
*   **白天关卡音乐**： 节奏明快、活泼的进行曲，随着僵尸波次增多，节奏和配器逐渐紧张。
*   **夜晚关卡音乐**： 旋律不变，但改为更为神秘、空灵的音乐变奏，加入铃铛、长笛等音色，节奏稍缓但保持悬疑感。
*   **屋顶关卡音乐**： 在白天主题音乐基础上，加入一些诙谐、滑稽的音乐元素，以配合屋顶的倾斜和花盆设定。
*   **关键事件音效**：
    *   **旗帜僵尸出现**： 一声响亮的号角声。
    *   **一大波僵尸即将来袭**： 紧张急促的警报声。
    *   **僵尸吃植物**： 持续的咀嚼声。
    *   **植物种植/攻击**： 各具特色的音效（如豌豆发射的“biu”声，向日葵产阳光的“叮咚”声）。
    *   **游戏胜利/失败**： 清晰独特的提示音与简短旋律。
